# 这只是一个简单的项目架子

## 技术栈

- React
- typeScript
- antDesign
- useReducer
- webpack
- Mockjs
- axios
- less

## 首先

- 你要需要安装一下项目的依赖，这里我优先推荐使用yarn

>npm install or yarn add or cnpm i

- 项目启动

>npm start


## 项目结构
```
project-root/
  README.md
  node_modules/
  package.json
  public/
  src/
    assets //静态文件资源
    component //公共组件
    config  //路由配置
    mock  //mock数据
    pages  //业务模块组件
    utils //工具函数库
```
